Introduction

Python is a high-level, interpreted programming language known for its simplicity and readability.

Created by Guido van Rossum and released in 1991, Python emphasizes code readability with its notable use of significant whitespace.

Python features a dynamic type system and automatic memory management.

print("Hello, World!")
Python Basics

Python uses indentation to define code blocks.

Variables in Python do not need explicit declaration.

Python supports multiple programming paradigms.

# Variable assignment x = 5 y = "Hello"
  • Easy to learn syntax
  • Dynamic typing
  • Interpreted language
Data Types

Python has several built-in data types:

  • Numbers (int, float, complex)
  • Strings
  • Lists
  • Tuples
  • Dictionaries
# Data type examples number = 42 text = "Python" list_example = [1, 2, 3]
Control Flow

Control flow is managed using if statements, loops, and function calls.

Python uses indentation to define blocks in control structures.

Common control flow statements include if, while, and for.

if x > 0: print("Positive") else: print("Non-positive") for i in range(5): print(i)
Functions

Functions are defined using the def keyword.

Python functions can return multiple values.

Arguments can have default values.

Functions are first-class objects in Python.

def greet(name="User"): return f"Hello, {name}!"
Object Oriented Programming

Python is a multi-paradigm programming language.

It supports object-oriented programming with classes and inheritance.

Classes are defined using the class keyword.

Objects are instances of classes.

class Dog: def __init__(self, name): self.name = name def bark(self): return f"{self.name} says woof!"
